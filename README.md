# tempmail

get your API key at [venumzmail.xyz](https://venumzmail.xyz), then replace the values in `main.py`:

```python
API_KEY = "your-key"
DOMAIN = "analgex.com"
```

install dependencies and run:

```bash
pip install requests
python main.py
```
